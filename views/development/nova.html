Nova Search

Nova is Nanodes front-end facing search engine. Nova offers the ability to search for specific queries as well as combine queries. In Range, Including, Within Folders, Only For.

Search Types

All options can be used simultaneously. Eg: Folders of Colour: #9cf, Size: '50kb - 50mb', Date: '05/06/2007 - 01/03/2021'
  • Within
    - Search Within All Directories in the section, or within the current directory.
  • By
    - Search by looking for specific colours or types.
  • In Range
    - Provides the option to only include items within the range of two dates, and / or two sizes.
  • Only For
    - Limits returned items to only Folders, or Files
  • Include
    - Tells Nova whether to read and match descriptions and previous names of items to the query.

Results

At current implementation, Nova does not index item data. As a result, a query requires loading the JSON structure of the required section. 'Within: Current' works differently by finding the parent and iterating through each child. Once the desired Nodes (items) are loaded, Nova checks each element against the queries. If called via the dropdown, Nova stops at 5 matches and returns. If more items are to be searched. Nova also provides a 'Load More' button.
