Void

Void is a hybrid between cloud storage and confidentiality. Void works by providing the user with USB style device to store the users file structure, information and pointers. The advantage being the full benefits of global access to storage, with the anonymity of saving files among everyone else without anything other than the users void device knowing who the files belong to. A Void device operates similarly to a key, where loss of a key and lack of a copy, prevents access to the safe.
