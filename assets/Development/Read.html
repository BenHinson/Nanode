Reading

Reading the User Specific JSON file and returning the information that was called for. One Call returns One Answer. Each call requires a UserID in the form of a cookie which is verified and required to open the users file. To try any examples login to your drive, press Ctrl + Shift + i and navigate to the console.

Note: Enter your own Item UUID's, to find what they are turn on 'Details', select the Item and copy the string just below the name.

  • UserID - Assigned xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
  • After - String "Contents", "Information", "CodexInfo", "CodexAfter: Notes, Videos, Music"
  • Path - String "Homepage", Item UUID, "Codex", "Bin"
  • Find - String "Span" or "File"

Call Contents

Call the directory of a folder or the Homepage, returning an array of information for each file within.
Example of SocketIO Emit to call directory of Folder: 'xxxxxxxx-1234-4568-9000-xxxxxxxxxxxx'
 
directoryPath = 
"Homepage" or Folder UUID (xxxxxxxx-1234-4568-9000-xxxxxxxxxxxx)


socket.emit ( 'directoryLocation' , { 
directoryPath
 } )
      

Server:
  returnInformation (
 userID ,  "Contents" ,  'xxxxxxxx-1234-4568-9000-xxxxxxxxxxxx' 
)
      
Return:
  Socket 'HomepageFolders' or 'DirectoryFolders' = 
[ 0: FOLDER UUID or FIRST SPAN CONTENTS, 1: CONTENTS or SECOND SPAN CONTENTS ]
 Server Side: UUID is Verified, UserJSON loaded, Object Found via ID, Item Security Checked, Contents Read. For Each Item inside Contents: [Name.Cur, Path.Cur, Type, Time.ModiT, Size, Tags, UUID] is pushed to returnArray and Emitted to the user via socket: HomepageFolders or DirectoryFolders

Call Information

Call meta-data about an item. Cannot call security answers.
Example of SocketIO Emit to call information of File: 'xxxxxxxx-1234-4568-9000-xxxxxxxxxxxx'
 
arrayOfInfo = 
["Name", "Type", "Size", "Time", "Share", "Tags"]


socket.emit ( 'ItemInformation', { "Path":
 "xxxxxxxx-1234-4568-9000-xxxxxxxxxxxx"
, "Wanted": 
arrayOfInfo
 } )
      

Server:
  returnInformation (
 userID, "Information", CallData.Path, CallData.Wanted
)
      
Return:
  Socket 'ReturnFolderInformation' = 
[0: {Cur: "Matrix", Pre: "ExampleName"}, 1: {isFi: true, isImg: "image/jpeg", mimeT: "image/jpeg"}, 2: 8438, 3: {ModiT: "9/5/2020"}, 4: {Group: "", Friend: ""}, 5: {Color: "rgb(0, 255, 79)"}]
